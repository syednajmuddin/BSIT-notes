## 📂 Course Modules & Core Subjects

### 1. 💻 Programming & Software Engineering
* 🌐 **Programming Introduction Reference:** [Study Fundamentals of Programming](https://beyond2013.github.io/ProgIntro)
* **Object-Oriented Programming (OOP):** Core object concepts, classes, and inheritance design slides.
* **Data Structures and Algorithms (DSA):** Implementation analysis and algorithm complexities.
* **Software Engineering (SE):** Detailed lecture series (Lecture 1 SE to Lecture 6 SE) covering software development lifecycles (SDLC) and requirements engineering.
* **Web Engineering & Technologies:** Advanced frameworks, front-end development models, and dynamic web architectures.

### 2. 🗄️ Database, Data Science & Advanced Analysis
* **Database Management Systems (DBMS):** Relational models, database design fundamentals, and introductory concepts.
* **Database Administration & Management:** Schema management, query optimization, and transaction handling.
* **Data Normalization:** Complete analysis guides for database tables management (1NF, 2NF, 3NF models).
* **Data Mining & Analytics:** Knowledge discovery techniques and algorithmic approaches.
* **Data Science Tools & Techniques:** Data-driven programming structures and statistical platforms.
* **Advanced Analysis of Algorithms:** Complex algorithmic patterns, dynamic programming, and computational complexity.
* **Probability and Statistics:** Mathematical models for continuous and discrete structures in computing.

### 3. 🔐 Cyber Security, Cryptography & Systems
* **Cryptography & Network Security:** Core encryption algorithms, secure protocols, and cryptography basics.
* **Digital Signatures & Certificates:** Infrastructure setups for public/private key verification and certificate authorities.
* **Firewalls:** Implementation strategies, packet filtering architecture, and network access control systems.
* **Operating Systems (OS):** Core theoretical structures and assignments based on standard literature (*Abraham Silberschatz*).

### 4. 🌐 Networking & Cloud Computing
* **Network Infrastructure:** Presentation guides covering OSI and TCP/IP reference architectures.
* **Network Protocols:** Handouts and documentation for standard communication protocols including ARP, ICMP, and FTP/TFTP.
* **Virtualization & Storage (VSS):** Hands-on architecture materials including the complete **UoB VSS 16-Week Lab Manual**, hypervisors case solutions, container handouts, and virtual networks blueprints.

### 5. 👔 Management, Professional Practices & Communication
* **IT Project Management (ITPM):** Agile structures, resource estimation models, and core management lecture series.
* **Technical & Paragraph Writing:** Specialized writing formats focusing on formal Memo writing, Memorandums, Press Releases, and the **7 Cs of Communication**.
* **Professional Practices:** Course modules addressing digital legal systems, ethical frameworks, and societal impacts of computing.
* **Business Development & Entrepreneurship:** Strategic slides focusing on opportunity recognition, product classification systems, feasibility analysis matrices, competitor analysis, consumer behavior, and complete marketing plan structures.
